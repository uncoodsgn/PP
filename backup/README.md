# Резервные копии

Архивы создаются скриптом из корня репозитория:

```bash
./scripts/backup.sh
```

Файлы вида `pp-backup-ГГГГММДД-ЧЧММСС.tar.gz` появляются в этой папке. В архив попадает проект целиком, кроме уже лежащих здесь `.tar.gz` и `.zip`, чтобы не раздувать новый архив старыми бэкапами.

Архивы по умолчанию **не коммитятся** (см. `.gitignore`). Храни копии на диске, в облаке или на внешнем носителе.
