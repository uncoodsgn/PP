# Cursor — контекст проекта PP

Статический портфолио-сайт: HTML, CSS, JavaScript. Язык интерфейса и части контента — русский; кейсы могут содержать английский текст.

## Структура

| Путь | Назначение |
|------|------------|
| `index.html` | Главная |
| `work.html`, `pet-projects.html` | Разделы работ |
| `case-*.html` | Страницы отдельных кейсов |
| `playground.html` | Песочница |
| `style.css`, `script.js` | Общие стили и скрипты |
| `img/` | Медиа |

## Cursor в репозитории

- **Правила агента:** `.cursor/rules/` (`.mdc` с frontmatter)
- **Навыки проекта:** `.cursor/skills/*/SKILL.md`
- **История изменений:** [CHANGELOG.md](./CHANGELOG.md)
- **Резервные копии:** [backup/README.md](./backup/README.md), скрипт `scripts/backup.sh`

## Ожидания к правкам

- Не ломать существующие классы и разметку без необходимости; новые страницы кейсов — по образцу `case-*.html`.
- Пути к ассетам относительные (`img/...`).
- После заметных изменений обновляй [CHANGELOG.md](./CHANGELOG.md) (раздел Unreleased или новая версия).
